# 🌱 Portal de Sostenibilidad – Coca-Cola

Proyecto web desarrollado para analizar la **economía circular y el ecodiseño** aplicado a un producto industrial.

El portal simula una plataforma tipo **Vinted**, donde se presenta el producto seleccionado (botella de Coca-Cola de 500 ml) junto con información sobre sostenibilidad, reciclaje y reducción del impacto ambiental.

## 🎯 Objetivo

Mostrar cómo un producto puede integrarse en un modelo de **economía circular**, aplicando principios de **ecodiseño** y alineándose con los **Objetivos de Desarrollo Sostenible (ODS)**.

## ♻️ Contenido de la página

El portal muestra información sobre:

- Ecodiseño del envase  
- Reciclabilidad del packaging  
- Reducción de emisiones de CO₂  
- Gestión sostenible del agua  
- Objetivos de Desarrollo Sostenible (ODS)

## 🛠 Tecnologías utilizadas

- HTML 
- CSS
